# 河野アラーム

「何アラームかけてるんですか？？勉強してください！！！」

## 概要

河野アラームは、勉強や作業に集中したい方のためのシンプルで効果的なアラームアプリです。設定した時間になると、「勉強してください！」という動画が再生され、作業に戻るよう促します。

## 特徴

- アラームモード：特定の時刻にアラームを設定
- タイマーモード：カウントダウンタイマーとして使用可能
- 河野の分身が画面上を浮遊する楽しいUI
- モバイル対応のレスポンシブデザイン

## 使い方

1. アラームモード：
   - 時刻を設定して「セット」ボタンをクリック
   - 設定時刻になると自動的にアラームが作動

2. タイマーモード：
   - タイマーモードに切り替え
   - キーパッドで時間を入力するか、プリセットボタンをクリック
   - 「セット」ボタンをクリックしてカウントダウン開始

## 技術仕様

- HTML/CSS/JavaScriptのみで実装
- フレームワークを使用せず、純粋なフロントエンド技術で構築
- ローカルでの実行が可能（サーバー不要）

## ファイル構成

- `index.html`: メインのHTMLファイル
- `kono.png`: 河野の画像
- `benkyo.mp4`: アラーム時に再生される動画

## 使用開始方法

1. リポジトリをクローンまたはダウンロード
2. `index.html`をブラウザで開く
3. アラームを設定して勉強に集中！
